## 4. Tablas con seq scans (correr en SQL Editor)

```sql
SELECT
  relname AS tabla,
  seq_scan,
  idx_scan,
  CASE WHEN seq_scan + idx_scan = 0 THEN 0
       ELSE round(100.0 * seq_scan / (seq_scan + idx_scan)::numeric, 2)
  END AS seq_pct,
  n_live_tup AS rows
FROM pg_stat_user_tables
WHERE n_live_tup > 100
ORDER BY seq_scan DESC
LIMIT 15;
```
